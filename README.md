## 🌐 About OSI-Model-Simulator

OSI-Model-Simulator helps users see the steps a message takes across the network layers. It covers everything from sending a request in the application layer to the physical transfer of signals. The simulator uses simple visuals and interactive controls to teach networking basics clearly.

Key points:

- Covers all seven OSI layers: Application, Presentation, Session, Transport, Network, Data Link, Physical.
- Shows both encapsulation (adding headers) and decapsulation (removing headers) processes.
- Interactive animations explain how data changes at each step.
- Built with React and TypeScript for smooth desktop performance.
- Cross-platform: runs on Windows and other desktops, plus a web version.

This tool suits students, teachers, technical support staff, and anyone interested in networking details.

## 🛠️ How to Use the OSI-Model-Simulator

The interface is clean and straightforward. Here is a basic guide to start learning with the app:

### Explore the Layers

The app displays all seven OSI layers in a vertical list. Each layer name is clickable. When you select one:

- See a simple explanation of what the layer does.
- Watch animations showing headers added or removed.
- Observe how the message changes shape or format.

### Simulate Encapsulation and Decapsulation

- **Encapsulation**: Data starts at the application layer with your message. The app shows how each layer adds its own header information.
- **Decapsulation**: The process reverses at the receiving side. Layers remove headers and pass the message upward.

You can toggle between these views using the controls at the top of the app.

### Step Through Message Flow

Use step buttons to move one step at a time or run the full animation automatically. The app pauses at each layer to highlight key points.

### Additional Features

- View TCP/IP concepts related to the OSI model.
- Test packet flow with example network data.
- Use dark or light mode for comfortable viewing.

The app helps visualize concepts that are often hard to grasp without practice.
